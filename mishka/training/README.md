# MISHKA Training Infrastructure

**AUTHORITATIVE:** Training infrastructure for MISHKA (Military-Grade Intelligence Support for Human Knowledge & Analysts)

## Directory Structure

```
training/
├── data/              # Training datasets
│   ├── raw/          # Raw data from sources
│   ├── processed/    # Processed training data
│   └── test/         # Test datasets
├── scripts/          # Training and data processing scripts
├── configs/          # Training configuration files
├── models/           # Model checkpoints and outputs
├── evaluation/       # Evaluation results and metrics
└── logs/            # Training logs
```
